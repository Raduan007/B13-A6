# DigiTools-platfrom

A modern digital marketplace built with React and Vite.  
Users can browse products, add to cart, and manage purchases easily.

---

## 🚀 Features

- 🛒 *Smart Cart*
  - Add/remove items instantly  
  - Real-time cart updates  

- ⚡ *Fast Loading*
  - Smooth UI with React Suspense  
  - No layout shift  

- 📱 *Responsive Design*
  - Works on mobile, tablet, desktop  

---

## 🛠️ Tech Stack

- React  
- Vite  
- Tailwind CSS  
- React Toastify  
- Lucide Icons  

---

## ❤️ Made With

Built using React + Vite + Tailwind CSS
